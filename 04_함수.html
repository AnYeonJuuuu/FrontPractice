4) 화살표 함수

        ES6에 추가됨

        * 파라미터 부분 변형 가능
        - 매개변수 0개 : () => {~~~}
        - 매개변수 1개 : x => {~~~}
        - 매개변수 2개 이상 : (x,y) => {~~~}

        * 리턴 처리
        - 로직이 여러 줄 일 때 : {중괄호} 필요
        - 로직이 한 줄 일 때 : () => return 리턴값; (중괄호 필요 없음, 리턴도 생략 가능)
        - 로직이 한 줄 일 때 : () => 리턴값;
